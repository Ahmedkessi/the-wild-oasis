# The Wild Oasis

A hotel management dashboard application built with React Query and Supabase.

This project was completed as part of Jonas Schmedtmann’s Ultimate React Course. I used the course project as a foundation to practice advanced React architecture, state management, authentication, routing, and backend integration.

## Features

* Cabin management
* Booking system
* Authentication & protected routes
* Dashboard analytics
* Dark mode
* Responsive UI

## Tech Stack

* React
* React Query
* Styled Components
* Advanced React Patterns
* Supabase
* React Router

## What I Learned

* Structuring large-scale React applications
* Server state management with React Query
* Authentication workflows
* Reusable component architecture
* Production-style project organization

## Future Improvements

* Add unit/integration tests
* Improve mobile responsiveness
* Refactor UI system
* Build additional admin features
